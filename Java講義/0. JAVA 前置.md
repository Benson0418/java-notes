PECS 原則

| 類型 | 主要行為 | 泛型邊界 |
|------|-----------|-----------|
| **Producer（生產者）** | 從集合中 **讀出元素**，用來計算、遍歷、操作等 | `? extends T` |
| **Consumer（消費者）** | 向集合中 **加入元素**，提供給其他地方使用 | `? super T` |
